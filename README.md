FeedPoint – Food Waste to Hunger Connect



About
FeedPoint is a simple web application that connects people who have extra food with those in need.  
It helps reduce food waste and supports people facing hunger.

Features
- User login & registration  
- Food donation system  
- Food request system  
- Simple and responsive UI  

Tech Stack
- HTML  
- CSS  
- JavaScript  

How It Works
1. User logs in  
2. Donor adds food details  
3. Receiver requests food  
4. Food is shared  

Note
This is a frontend-only college project (no backend/database).
